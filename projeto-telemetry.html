Equipe Draconis | Business Intelligence

Telemetry UI

Dashboard de visualização de dados para otimização de performance e análise de mercado de drones.

O Contexto

A equipe universitária Draconis precisava de uma maneira mais eficiente de interpretar os dados de voo de seus drones competitivos, além de estruturar informações sobre o mercado e a intenção de compra do produto. Dados brutos de telemetria são difíceis de analisar rapidamente durante uma operação.

A Solução

O projeto consistiu em desenvolver uma interface de telemetria (UI) focada em Business Intelligence. O painel traduz métricas complexas (como altitude, velocidade, e eficiência de bateria) em gráficos visuais intuitivos. A solução também engloba a coleta estruturada de dados do público, integrando formulários de intenção de compra para a equipe de marketing.

Meu Papel

Trabalhei na intersecção entre a análise de dados e a interface do usuário. Utilizei Python para processar os dados recebidos e bibliotecas de visualização para renderizar os painéis interativos. O foco foi garantir que a tomada de decisão da equipe fosse guiada por dados precisos e em tempo real.
