Retro 80s Digital Clock

Um relógio digital que resgata a estética neon clássica através de programação moderna.

A Ideia

Como um exercício criativo unindo lógica de programação e design, decidi desenvolver um relógio/alarme digital com uma forte estética retro dos anos 80. O desafio era recriar o brilho, a fonte segmentada e a interface nostálgica utilizando apenas código.

O Desenvolvimento

O projeto foi inteiramente construído utilizando Python. Explorei bibliotecas de interface gráfica para desenhar os dígitos de sete segmentos e aplicar efeitos de luz e sombra que emulam displays de LED antigos. O foco principal foi garantir uma UI/UX suave e imersiva, sem comprometer a performance do relógio.

Funcionalidades

Além de exibir a hora com precisão matemática, a aplicação inclui um sistema de alarme configurável e opções para alternar entre diferentes esquemas de cores típicos do movimento synthwave (ciano, magenta e laranja neon).
