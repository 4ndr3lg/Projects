Estresse Hídrico e Servidores

Análise de dados explorando o custo ambiental oculto por trás do processamento em nuvem.

O Problema

O crescimento exponencial do uso de Inteligência Artificial e serviços em nuvem exige a construção de Data Centers massivos. O que muitas vezes é ignorado é a quantidade gigantesca de água necessária para resfriar esses servidores. O projeto nasceu da necessidade de quantificar e expor esse estresse hídrico.

A Pesquisa

Realizei um cruzamento de dados públicos sobre o consumo de água de grandes empresas de tecnologia e os mapas de escassez hídrica global. O objetivo foi criar uma narrativa baseada em dados que aliasse o desenvolvimento de software com a responsabilidade ambiental e o ambientalismo moderno.

Resultados e Divulgação

Os dados estruturados foram preparados para serem transformados em conteúdo de alto impacto visual (como peças para o Instagram). O projeto prova que a engenharia de software e a análise de dados são ferramentas poderosas para a conscientização sobre as mudanças climáticas.
